# Discord Bot Izken (In Development)


## Installation

Make sure to have [Node.js](https://nodejs.org/) installed on your system.

1. Clone this repository:
   ```bash
   git clone https://github.com/NoException-exe/DiscordJs.git
   ```

2. Navigate to the bot's directory:
   ```bash
   cd DiscordJs
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

## Configuration

1. Rename the `config.env` file to `.env`.
2. Open the `.env` file in a text editor and enter your Discord bot credentials.

## Execution

Run the bot using the following command:

```bash
node src/bot.js
```
